# Знакомство с IaaS, PaaS, SaaS сервисами в облаке на примере Amazon Web Services (AWS). Создание сервисной модели.

IaaS (Infrastructure as a Service)
IaaS — это модель облачных вычислений, при которой провайдер предоставляет базовые инфраструктурные ресурсы, такие как виртуальные серверы, хранилища данных и сети, с оплатой по мере использования. Пользователь самостоятельно управляет операционной системой, приложениями и данными, а провайдер отвечает только за физическое оборудование. Примеры: Amazon EC2, Google Compute Engine.​

PaaS (Platform as a Service)
PaaS предлагает готовую платформу для разработки, тестирования и развертывания приложений, включая инструменты, middleware и runtime-среду. Пользователь фокусируется на коде, не заботясь об инфраструктуре или ОС, которую управляет провайдер. Примеры: Google App Engine, Heroku.​

SaaS (Software as a Service)
SaaS — это готовое программное обеспечение, доступное через интернет без установки, с полным управлением со стороны провайдера (инфраструктура, приложения, обновления). Пользователи просто используют сервис в браузере. Примеры: Google Workspace, Microsoft Office 365.​
