# Лабораторная работа 3

В ходе работы был написан скрипт, выполняющий следующие функции:
- Динамическое выделение портов: Каждый новый инстанс запускается на инкрементируемом порту (9000, 9001...).
- Персистентность данных: Использовано монтирование томов (Volumes). Данные каждого пользователя сохраняются локально в директории DATA_BASE_PATH/{instance_id}, что позволяет сохранять файлы после перезапуска контейнера.
- Безопасность: Автоматическая генерация уникальных ключей доступа через uuid.
